Badges
Badges are simple and basic components that are used to display an indicator or count a number.
Avatar Badges
Use avatars along with badges to display more details like count or status like online or offline. Use only "badge" class to display number as badge and use "badge-offline" and "badge-online" to display status. You can use it with both regular rectangular and circular avatar
